# 2026-05-20 Worklog

## Blue Dragon Thor Debug Loop

- Added `arm64_speed_profile_thread_snapshot` for the A64 speed profiler.
  - Generated A64 prologues now store the current guest function and incoming
    guest return address in `A64BackendContext`.
  - Speed-profile intervals can log each XThread's `last_fn`, `last_ret`,
    LR/CTR/R1/R3/R4.
  - The snapshot path retries the processor debug lock for short lock pressure
    and reports `last_global_owner_sys_tid` plus PPC global-lock breadcrumbs if
    it still cannot sample.
- Added `Processor::TryQueryThreadDebugInfos` so debug sampling does not block
  the profiler thread indefinitely.
- Added lightweight PPC global-lock owner breadcrumbs in `ppc_frontend`.
- Added generic global critical-region last-owner breadcrumb for profiler
  skip diagnostics.
- Added Android all-files debug access:
  - Manifest now declares `MANAGE_EXTERNAL_STORAGE`.
  - `tools/thor/thor_build.ps1` reapplies the `MANAGE_EXTERNAL_STORAGE` app-op
    after install/deploy when a device serial is provided.
- Changed the Thor debug launcher default to `mount_cache=true`.

## Tooling Changes

- Added `tools/thor/thor_xenia_debug.ps1 -Mode LaunchBlueDragonTitleCapture`.
  - Defaults to 45 seconds.
  - Avoids auto START/A input and the forced signed 10:10:10:2 Vulkan fallback
    used by the speed lane.
  - Adds optional `-TitleScreenshotSeconds` for timed screenshots; default is
    `0` because mid-run screencaps can perturb the timing-sensitive title route.
  - Records `title_screenshot_seconds` in capture metadata.
- Added `arm64_speed_profile_thread_snapshot_on_idle`.
  - Routed through Android intent extras and `tools/thor/thor_xenia_debug.ps1`
    as `-Arm64SpeedProfileThreadSnapshotOnIdle`.
  - It waits for at least one active A64 profile interval, then logs one thread
    snapshot only if a later interval goes idle.
  - This is intended for black-idle captures where always-on thread snapshots
    change early title-route timing.
- Build/deploy validation for the idle snapshot lane passed with
  `tools/thor/thor_build.ps1 -Mode FullDeploy -DeviceSerial c3ca0370`.
- Idle snapshot lane proof: `scratch/thor-debug/20260520-012015-*`.

  - Route reached the visible `press START` title screen and counters stayed
