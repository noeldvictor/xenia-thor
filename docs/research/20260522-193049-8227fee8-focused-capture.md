## Tooling Patch

Filtered function dumps now use warning-level logging in
`src/xenia/cpu/ppc/ppc_translator.cc`, so explicit
`-DisassembleFunctionFilter` captures can be visible in the normal `log_level=1`
speed lane. `tools/thor/thor_xenia_debug.ps1` now records
`disassemble_functions` and `disassemble_function_filter` in capture metadata
and prints the active function filter in the capture summary.

Validation before the patched device run:

```text

powershell -NoProfile -ExecutionPolicy Bypass -File tools\thor\thor_build.ps1 -Mode NativeCore
powershell -NoProfile -ExecutionPolicy Bypass -File tools\thor\thor_build.ps1 -Mode FullDeploy -DeviceSerial c3ca0370
```

The patched filtered run was `scratch/thor-debug/20260522-192429-*`, APK SHA
`89086669EF6CC19A028049FBF5957827CF7CFA97C85F0083021A739B9C16FFAF`. Metadata

black-idled before `82282490`/`8227FEE8` compiled far enough to emit HIR or

This does not prove the warning-level dump patch unsafe, because the route has
known nondeterministic black-idles. It does mean the exact filtered capture
should not be rerun unchanged.

## Decision

Keep the filtered-dump tooling patch. It only changes explicitly requested
filtered dumps, and it avoids the much heavier `log_level=0` route.

Do not start a `8227FEE8` peephole from body-time alone. First get the HIR shape
with a route-safe lane.

Next useful options:

- Run a patched same-APK control without `-DisassembleFunctionFilter` and with
  delayed `-Arm64SpeedProfileBodyTimeFilter 82282490,8227FEE8` to prove route
  safety after the tooling patch.
- If the control reaches opening, run one filtered capture with the same delayed
  body-time setup and compare against a second control.
- If filtered capture remains route-sensitive, use offline guest PPC/Ghidra
  analysis for `8227FEE8` while keeping device HIR capture gated by a control
  sandwich.

Note for future parser work: body-time address filters can include nearby or
enclosing functions in some lanes. Use exact function-start filters for
block/call-edge instrumentation when target isolation matters.
