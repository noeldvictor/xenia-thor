## Decision

Do not implement a local `822824F0-82282528` vector-load codegen patch yet. The
span has about `2.0M` approximate exclusive ticks, but the useful vector value
is assembled across the `822824F0` / `8228252C` boundary. A local peephole that
only sees the two loads could easily miss the `lvrx` zero path, the `v[13]`
producer, or the later `vor/extract/splat/vmaddfp` consumer chain.
